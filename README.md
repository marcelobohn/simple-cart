# SimpleCart

[![Build Status](https://travis-ci.org/marcelobohn/simple-cart.svg?branch=master)](https://travis-ci.org/marcelobohn/simple-cart)

Install

```
git clone https://github.com/marcelobohn/simple-cart.git
bundle
rake db:create
rake db:seed
```

Tests

```
rspec
```

Run app

```
rails s
```
